# Todo Application

A minimal and efficient Todo List app built using **React** and styled with **Tailwind CSS**.

### Features:
- Add, delete, and mark todos
- Fully responsive layout
- Clean design using Tailwind classes

### Tech Stack:
- React
- Tailwind CSS
- JavaScript
